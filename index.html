Zapier Workflow
dgMarket RSS → Gmail digest
  1. 1
    Go to zapier.com → Create Zap → Trigger: RSS by Zapier
  2. 2
    Paste the dgMarket RSS URL from the panel below into the Feed URL field
  3. 3
    Add Filter: only continue if Title contains: supervision, AMO, consultant, advisory, PMO, étude, feasibility, port
  4. 4
    Action 1 — Gmail → Send Email: To: louai@skyrisecapitalpm.com · Subject: [Tender Alert] {{Title}} · Body: {{Link}} — {{Description}}
  5. 5
    Action 2 (optional) — Google Sheets → Append Row: log Title, Country, Deadline, URL for tracking
  6. 6
    Set schedule: every 6 hours. Name and enable the Zap.
🔄
Make (Integromat)
Multi-source · AI scoring scenario
  1. 1
    New Scenario → Add module: RSS Feed — add one per source (dgMarket, UNDP, Devex)
  2. 2
    Add Router to split by source and apply sector/keyword filters per feed
  3. 3
    Add HTTP → Make a Request — POST to Claude API with the scorer payload (see below)
  4. 4
    Add JSON → Parse JSON to extract the score and reason from Claude's response
  5. 5
    Add Filter: only pass tenders with score ≥ 75
  6. 6
    Action: Gmail digest + Airtable row for live dashboard feed
  7. 7
    Schedule every 4 hours. Activate scenario.
