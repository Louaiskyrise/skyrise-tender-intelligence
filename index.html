Zapier Workflow
dgMarket RSS → Gmail digest
  1. 1
    Go to zapier.com → Create Zap → Trigger: RSS by Zapier
  2. 2
    Paste the full dgMarket RSS URL from the panel below into the Feed URL field
  3. 3
    Add Filter: only continue if Title contains: supervision, AMO, consultant, advisory, PMO, étude, feasibility, port
  4. 4
    Action 1 — Gmail → Send Email: To: louai@skyrisecapitalpm.com · Subject: [Tender Alert] {{Title}}
  5. 5
    Action 2 (optional) — Google Sheets → Append Row: log Title, Country, Deadline, URL
  6. 6
    Schedule: every 6 hours. Enable Zap.

Make (Integromat)
Multi-source · AI scoring scenario
  1. 1
    New Scenario → RSS Feed module — one per source (dgMarket, UNDP, Devex)
  2. 2
    Add Router to split by source and apply keyword filters per feed
  3. 3
    Add HTTP → Make a Request — POST to Claude API with scorer payload below
  4. 4
    Add JSON → Parse JSON — extract score and reason
  5. 5
    Add Filter: only pass tenders with score ≥ 75
  6. 6
    Action: Gmail digest + Airtable row
  7. 7
    Schedule every 4 hours. Activate scenario.
